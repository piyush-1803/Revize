# Revize

Revize is a smart revision planner that schedules your study sessions using the forgetting curve, helping you review topics at the right time for better long-term retention.

## What it does

Revize helps you:
- Plan revision sessions based on memory decay patterns.
- Review topics before they are likely to be forgotten.
- Build a consistent spaced-repetition study habit.
- Track what you studied and when it should be revised next.

## Why Revize

Most people forget new information quickly if they do not revise it soon. Revize solves this by automatically spacing out revision sessions so your memory stays stronger over time [web:2][web:8].

## Features

- Forgetting-curve-based revision scheduling.
- Smart reminders for upcoming revision sessions.
- Topic-wise study tracking.
- Spaced repetition support.
- Simple and focused interface for learners.

## How it works

1. Add a topic or chapter you want to revise.
2. Complete your first study session.
3. Revize schedules the next revision based on the forgetting curve.
4. Keep revising at the right intervals to improve retention.

## Ideal for

- Students preparing for competitive exams.
- Learners who want to remember more for longer.
- Anyone using spaced repetition for study.
